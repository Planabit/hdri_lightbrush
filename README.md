# HDRI Editor Blender Addon

Ez a projekt egy Blender addon fejlesztésére szolgál, amely HDRI képek szerkesztését teszi lehetővé.

## Fejlesztési lépések
- Python alapú Blender addon szerkezet
- Verziókövetés Git-tel

## Használat
1. Másold a projekt mappáját a Blender `addons` könyvtárába, vagy használd a Blender beépített addon telepítőjét.
2. Engedélyezd az addont a Blenderben.

## Követelmények
- Blender (ajánlott: legfrissebb verzió)
- Python 3.7+

## Fejlesztés indítása
1. Klónozd vagy töltsd le ezt a repót.
2. Nyisd meg a projektet VS Code-ban.
3. Fejleszd az addont a `src` mappában.
